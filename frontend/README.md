## ✨ Features
* [cite_start]**🧠 Viral Signal Fusion:** Combines audio energy (Librosa), semantic profundity (Gemini 1.5 Flash), and keyword density (Whisper) to score virality. [cite: 14, 121, 156]
* [cite_start]**🎯 Smart-Crop (9:16):** Uses MediaPipe Face Detection to track speakers in real-time, converting horizontal footage into vertical reels flawlessly. [cite: 16, 160]
* [cite_start]**💬 Karaoke Captions:** Dynamic, timed word-level overlays to maximize scroll-stopping engagement. [cite: 12, 17, 289]
